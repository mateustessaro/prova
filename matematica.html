Δ = 1² - 4.1.(-20)
Δ = 1 + 80
Δ = 81
Como Δ > 0, então a equação possui duas raízes reais.
x = (1+-√(81))/(2)
x = (1+-9)/(2)
Portanto, as raízes são:
x' = (-1+9)/(2) = 4
x'' = (-1-9)/(2) = -5
Como queremos x² + x - 20 ≤ 0, então os inteiros estão entre -5 e 4.
São eles: -5, -4, -3, -2, -1, 0, 1, 2, 3, 4.
